# Digital Launch — Деплоймент статичного вебдодатку

Навчальний проєкт (лабораторна робота) з розгортання та хостингу сучасного промо-сайту на безкоштовному сервері Render.com з використанням системи контролю версій Git та репозиторію GitHub.

## Опис проєкту
Цей проєкт є повністю статичним вебсайтом, створеним на базі чистих вебтехнологій без використання додаткових фреймворків чи серверного бекенду.

**Основні характеристики:**
- **Технологічний стек:** HTML5, CSS3, JavaScript ES6.
- **Дизайн:** Темна/світла теми (збереження в LocalStorage), адаптивна кросбраузерна верстка, анімація при прокручуванні сторінки (Intersection Observer API), інтерактивні FAQ-акордеони, спливаючі повідомлення (Toast Notifications).
- **Платформа розгортання:** Render.com (тип ресурсу: Static Site).
